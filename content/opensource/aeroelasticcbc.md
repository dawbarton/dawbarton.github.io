---
title: "AeroelasticCBC.jl"
---

Code associated with the research project on hybrid modelling and nonlinear dynamics in aeroelastic systems. The repository contains implementations of control-based continuation (CBC) applied to aerofoil flutter experiments, including the scientific machine learning (SciML) methods used to augment mechanistic models with data-driven corrections.

This code underpins the published work on using universal differential equations trained via CBC data to produce accurate low-dimensional models of flutter behaviour — see [Beregi, Barton, Rezgui, Neild (2022)](https://research-information.bris.ac.uk/en/publications/using-scientific-machine-learning-for-experimental-bifurcation-an).

Source code available from [AeroelasticCBC.jl](https://github.com/dawbarton/AeroelasticCBC.jl) on GitHub.
