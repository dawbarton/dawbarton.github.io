---
title: "NumericalContinuation.jl"
---

A numerical continuation package for Julia, written in the style of [COCO](https://sourceforge.net/projects/cocotools/) (the Computational Continuation Core toolbox for MATLAB). Numerical continuation is the core computational tool for tracing families of solutions (periodic orbits, equilibria, bifurcation points) of nonlinear systems as parameters vary, and for constructing bifurcation diagrams.

The package is designed to be composable: different problem types (ODEs, DDEs, hybrid systems, manifold constraints) can be combined within the same continuation run. Development is ongoing as part of the research programme in [control-based continuation](../research/background/) and nonlinear dynamics.

Source code available from [NumericalContinuation.jl](https://github.com/dawbarton/NumericalContinuation.jl) on GitHub.
